# tvheadend-channel-importer

### Usage

```
$ php bin/tvheadend-channel-importer.php --help
 bin/tvheadend-channel-importer.php                                                                                              

--help
     Show the help page for this command.


--source-network-filter <argument>
     The network the source channel list should be filtered by before importing


--source-tvheadend-hostname <argument>
     Required. The hostname where tvheadend is running


--source-tvheadend-http-port <argument>
     The tvheadend HTTP port


--source-tvheadend-password <argument>
     The tvheadend password


--source-tvheadend-username <argument>
     The tvheadend username


--target-network-name <argument>
     Required. The name of the network the channels will be imported to


--target-tvheadend-hostname <argument>
     Required. The target hostname where tvheadend is running


--target-tvheadend-http-port <argument>
     The target tvheadend HTTP port


--target-tvheadend-password <argument>
     The target tvheadend password


--target-tvheadend-username <argument>
     The target tvheadend username
```
